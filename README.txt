Реализация простого TDMA (Time Division Multiple Access):
	-Каждому датчику выделяется свой временной слот для передачи.
	-Сервер (RPi) периодическ шлет широковещательный пакет-beacon ("начало цикла").
	-Все датчики получают этот beacon и синхронизируются по нему.
	-Каждый датчик знает свой порядковый номер (ID) и вычисляет свое время для передачи: beacon_time + (ID * slot_duration).
	-Датчик "просыпается" строго в свой слот, отправляет данные и снова "засыпает".

Формат принимаемого сообщения: st beacon_time fn  

В Test-Raspberry находится терминальная программа для отправки и приема данных с датчиков
